# FastAPI 실습 예제
- 이 디렉토리의 예제 코드들은 FastAPI(1) 강의 내용과 연결됩니다.
- 강의를 먼저 들은 후, 코드를 직접 실행하며 학습하는 것을 추천합니다.

## 예제 파일 목록
- `01_simple_webserver.py`: 가장 기초적인 웹 서버 구동
- `02 ~ 04`: Path/Query Parameter 실습
- `05 ~ 08`: Request Body, Form, File 처리
- `09_validation.py`: Pydantic을 이용한 데이터 검증
- `10 ~ 15`: 설정, 라우터, 예외 처리, 백그라운드 작업 등 심화 내용
